<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">胸部 X 射线数据集和模型的库。</font><font style="vertical-align: inherit;">包括预先训练的模型。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">TorchXRayVision 是一个开源软件库，用于处理胸部 X 射线数据集和深度学习模型。</font><font style="vertical-align: inherit;">它为大量公开可用的胸部 X 射线数据集提供了通用接口和通用预处理链。</font><font style="vertical-align: inherit;">此外，该库还提供许多具有不同架构、针对不同数据组合进行训练的分类和表示学习模型，用作基线或特征提取器。</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对于解决临床问题的研究人员来说，从头开始训练模型是浪费时间。</font><font style="vertical-align: inherit;">为了解决这个问题，TorchXRayVision 提供了预先训练的模型，这些模型在大量数据上进行训练，并实现 1) 大型数据集的快速分析 2) 特征重用以进行少量学习。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对于开发算法的研究人员来说，使用多个外部数据集稳健地评估模型非常重要。</font><font style="vertical-align: inherit;">与每个数据集关联的元数据可能差异很大，这使得很难将方法应用于多个数据集。</font><font style="vertical-align: inherit;">TorchXRayVision 以统一的方式提供对许多数据集的访问，以便可以用一行代码交换它们。</font><font style="vertical-align: inherit;">这些数据集还可以合并和过滤，以构建用于研究泛化的特定分布变化。</font></font></li>
</ul>

<div class="markdown-heading" dir="auto"><h2 tabindex="-1" class="heading-element" dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">分配班次工具（</font></font><a href="https://github.com/mlmed/torchxrayvision/blob/master/scripts/xray_datasets-CovariateShift.ipynb"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">演示笔记本</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">）</font></font></h2><a id="user-content-distribution-shift-tools-demo-notebook" class="anchor-element" aria-label="永久链接：分发转移工具（演示笔记本）" href="#distribution-shift-tools-demo-notebook"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a></div>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">该类</font></font><code>xrv.datasets.CovariateDataset</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">采用两个数据集和两个表示标签的数组。</font><font style="vertical-align: inherit;">样本将按照每个站点所需的图像比例返回。</font><font style="vertical-align: inherit;">这里的目标是模拟协变量转变，使模型专注于不正确的特征。</font><font style="vertical-align: inherit;">然后，验证数据中的转变可能会逆转，从而导致泛化性能出现灾难性失败。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ratio=0.0 表示 d1 中的图像将具有正标签。ratio=0.5 表示 d1 中的图像将具有一半的正标签。ratio=1.0 表示 d1 中的图像将没有正标签。</font></font></p>
